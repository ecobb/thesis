fmplib.unit7.audio_to_ssm(snd, cens_filt_len, cens_ds, fft_len=4096)¶

Create a self-similarity matrix (SSM) from an audio signal by using CENS as the feature set.

Parameters

  • snd – the audio signal

  • cens_filt_len – the CENS filter length

  • cens_ds – the CENS downsample rate

  • fft_len – the length of the FFT used in creating the initial chromagram, before CENS is applied.

Returns

a tuple (ssm, ff) where ssm is the self-similarity matrix, and ff is its the feature rate

fmplib.unit7.calc_prf(est, ref, tau)¶

Given estimated values and reference values, and based on the match criteria of |e - r| < tau, calculate Precision, Recall, and F-measure.

Parameters

  • est – an array of estimate values

  • ref – an array of reference values

  • tau – the distance criteria to determine a match

Returns

the tuple (P, R, F) with values for Precision, Recall, and F-measure.

fmplib.unit7.column_derivative(m, k_len)¶

Create a derivative kernel of length k_len and apply it to input matrix m. The kernel is slid from left to right. At each step, compute the sum of element-wise products for each row, thus creating a column vector. The length of this column vector is the value of the output novelty function at each point.

Parameters

  • m – the input matrix

  • k_len – the length of the derivative kernel

Returns

an array of values representing a novelty curve for matrix m.

fmplib.unit7.diagonal_novelty(ssm, cb_len)¶

Slide a checkerboard kernel along the main diagonal of a self-similarity matrix to produce a boundaries novelty curve. At each point, compute the sum of the element-wise products of the SSM and the checkerboard.

Parameters

  • ssm – an (N x N) self-similarity matrix

  • cb_len – the size of the checkerboard to use

Returns

an array of length N, the boundaries novelty curve.

fmplib.unit7.diagonal_slide(ssm, kernel)¶

Slide the given kernel along the main diagonal of a self-similarity matrix to produce a novelty curve. At each point, compute the sum of the element-wise products of the SSM and the kernel.

Parameters

  • ssm – an (N x N) self-similarity matrix

  • kernel – a (L x L) kernel

Returns

an array of length N, the novelty curve.

fmplib.unit7.make_checkerboard(cb_len, gaussian=True, normalize=True)¶

Create a Gaussian checkerboard kernel with positive values on the main diagonal and negative values on the off-diagonal.

Parameters

  • cb_len – the full length of the checkerboard. Value should be even.

  • gaussian – if True, make output a Gaussian-tapered checkerboard. Otherwise, make a binary checkerboard.

  • normalize – if True, normalize output to sum to 1.

Returns

a matrix of shape (cb_len, cb_len).

fmplib.unit7.pos_neg_split(est, ref, tau)¶

Evaluate estimated values against reference values, returning positive and negative match results for each set. Positive matches are based on the criteria of |e - r| < tau. The estimates array and reference array are sorted into two lists: matches (positives) and non-matches (negatives).

Parameters

  • est – an array of estimate values

  • ref – an array of reference values

  • tau – the distance criteria to determine a match

Returns

a tuple of four arrays (est_pos, est_neg, ref_pos, ref_neg) where each array is the positive or negative subset of the corresponding input array. By definition, len(est_pos) == len(ref_pos)

fmplib.unit7.thresh_and_scale(m, tau)¶

Apply thresholding and scaling to the input matrix.

Parameters

  • m – the input matrix. All values must be <= 1.

  • tau – the threshold value in the range [0,1]

Returns

a matrix of the same dimensions as m. All values of m < tau are set to 0. Then all values from [tau, 1] are expanded to the range [0, 1].
