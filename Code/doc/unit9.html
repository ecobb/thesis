fmplib.unit9.hpss(x, fft_len=2048, h_filt_len=13, p_filt_len=21, soft_mask=True)¶

Perform harmonic / percussive source separation using soft masks

Parameters

  • x – the input audio signal

  • fft_len – the fft length used to create the spectrogram

  • h_filt_len – the window length used for harmonic median filtering

  • p_filt_len – the window length used for percussive median filtering

  • soft_mask – Use a soft mask (Wiener filtering) if True. Otherwise, use a binary mask.

Returns

a tuple of (x_h, x_p), where x = x_h + x_p and each represents the harmonic / percussive components of x.

fmplib.unit9.hpss_masks(spec_h, spec_p, soft_mask=True)¶

Create soft or binary masks from harmonic and percussive filtered spectrograms

Parameters

  • spec_h – the harmonic spectrogram

  • spec_p – the percussive spectrogram

  • soft_mask – Create soft masks (Wiener filtering) if True. Otherwise, create binary masks.

Returns

the tuple (mask_h, mask_p), where mask_h and mask_p are the harmonic and percussive masks.

fmplib.unit9.istft(X, hop_size, centered=True, ignore_windowing=False)¶

Perform the inverse short time Fourier transform. Assumes that the STFT was created with a hanning window.

Parameters

  • X – the complex spectrum, assumed to be the first half + 1 of the full complex spectrum

  • hop-size – the hop size used in the original stft

  • centered – if the original stft was created with centered windows.

  • ignore_windowing – do not undo the effects of windowing.

Returns

a 1d-array of the reconstructed signal

fmplib.unit9.make_nmf_note_model(fs, fft_len, pitch)¶

Create an NMF-suitable template vector that models the harmonics of a note, where a set of bins around the note’s harmonics are given the value 1/h. The width is set as +/- 0.5 semitones around the fundamental frequency.

Parameters

  • fs – the sample rate used to create the note template

  • fft_len – the fft length for this template. The output vector length is therefore fft_len // 2 + 1

  • pitch – midi pitch of fundamental frequency for the note

Returns

a vector of frequency bins with energy distributed as 1/h across harmonics of the given note.

fmplib.unit9.make_nmf_template(fs, fft_len, pitches)¶

Create an NMF-suitable template matrix for a collection of pitches. See make_nmf_note_model()

Parameters

  • fs – the sample rate used to create the note templates

  • fft_len – the fft length for this template. The number of rows is K = fft_len // 2 + 1

  • pitches – a list of midi pitches, the length of which is R, number of columns.

Returns

W, a K x R templates matrix

fmplib.unit9.mask_and_reconstruct(W, H, Hsub, stft, hop_size)¶

Create a NMF-based soft mask, apply it to a STFT, and reconstruct the signal using ISTFT

Parameters

  • W – The templates matrix

  • H – The activations matrix

  • Hsub – a filtered activations matrix, containing a subset of activations for mask-creation

  • stft – The original STFT to be masked

  • hop_size – The hop size used to create the STFT

Returns

a tuple of the mask and the post-masking reconstructed signal
