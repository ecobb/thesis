class fmplib.unit8.FingerprintDB¶

Manages creating a fingerprint database and performing queries using the audio identification algorithm.

add(fp)¶

Add an audio file to the database

Parameters

fp – file path for the audio file (must be a 22,050Hz wave file)

ff()¶

returns the feature frequency

query(audio, num_matches=1)¶

Test an audio query against the database, returning the given number of matches

Parameters

  • audio – the audio signal, sample at 22,050Hz

  • num_matches – number of matches to return

Returns

the tuple (matches, query_points), where matches is a list of (song_name, location, num_matched_pts) and query_points is number of constellation points in the query

class fmplib.unit8.MatchDB¶

Manages creating a database and performing queries using the audio matching algorithm.

add(fp)¶

Add an audio file to the database

Parameters

fp – file path for the audio file (must be a 22,050Hz wave file)

ff()¶

returns the feature frequency

query(audio, num_matches, return_match_fn=False)¶

Test an audio query against the database, returning a given number of matches

Parameters

  • audio – the audio signal, sample at 22,050Hz

  • num_matches – number of matches to return

  • return_match_fn – if True, also return the matching function d_dtw

Returns

results or (results, d_dtw), depending on return_match_fn. results is a list of length num_matches where each item is (song_name, start_time, end_time, cost)

fmplib.unit8.constellation_slice(points, n_start, n_end)¶

Return a subset of points from a constellation

Parameters

  • points – the input constellation, a P x 2 matrix of points.

  • n_start – filter out points with n < n_start

  • n_end – filter out points with n >= n_end

Returns

a P x 2 matrix of points that match the specified criteria

fmplib.unit8.dtw_match(C, num_matches)¶

Performs the matching variant of dynamic time warping for the cost matrix C = query x database.

Parameters

  • C – the cost matrix, an N x M where N is the length of the query and M is the length of the database

  • num_matches – number of matches to return

Returns

a tuple of D, the accumulated cost matrix, and a list of matches. Each match is (start_loc, end_loc, cost)

fmplib.unit8.make_constellation(snd, fft_len, hop_size, max_bin, zone_width=60, zone_height=40)¶

Convert audio data into a constellation of points (fingerprint).

Parameters

  • snd – waveform data

  • fft_len – fft length for creating the STFT

  • hop_size – fft hop size for creating the STFT

  • max_bin – only consider STFT from 0 up to this bin

  • zone_width – the width of the rectangular patch used in picking a local peak

fmplib.unit8.slide_match(db, query)¶

Create a matching function of a query against a database by sliding the query against the database, returning the number of intersecting points at each shift value

Parameters

  • db – the points of the database

  • query – the points in the query

Returns

the matching function, a 1D array, whose values correspond to the number of intersecting points between db and query at each shift of query from 0 to the length of db.
