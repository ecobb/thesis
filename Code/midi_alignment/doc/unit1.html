fmplib.unit1.make_additive_tone(amps, freqs, dur, fs)¶

Creates a complex tone by using additive synthesis, that is, summing a set of sine tones with fixed frequencies and amplitudes.

Parameters

  • amps – array of amplitudes applied to each sinusoid

  • freqs – array of frequencies (in Hertz) for each sinusoid

  • dur – duration in seconds of generated tone

  • fs – audio sampling rate

Returns

np.array

fmplib.unit1.make_envelope(db1, db2, L)¶

Creates a volume envelope curve that is linear in decibels.

Parameters

  • db1 – initial value in decibels

  • db2 – final value in decibels

  • L – length (in samples) of signal

Returns

np.array

fmplib.unit1.make_sine_tone(amp, freq, dur, fs)¶

Creates a pure sine tone of a fixed frequency and length.

Parameters

  • amp – amplitude, typically > 0

  • freq – frequency in Hertz

  • dur – duration in seconds

  • fs – audio sampling rate

Returns

np.array

fmplib.unit1.make_tone_series(amp, freqs, dur, fs)¶

Creates a series of pure sine tone, each of a fixed length, concatenated together. Also adds a bit of fade-in / fade-out to avoid pops between tones.

Parameters

  • amp – array of amplitudes, one per tone, or scalar value for all tones.

  • freqs – array of frequencies in Hertz

  • dur – duration in seconds of each tone

  • fs – audio sampling rate

Returns

np.array

Parameters

p – MIDI pitch value (can be float)

frequency (float)
