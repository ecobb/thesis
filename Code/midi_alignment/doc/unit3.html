fmplib.unit3.stft(x, win_len, hop_size, zp_factor=1, window=None, centered=True)¶

Return the Short-Time-Fourier-Transform.

Parameters

  • x – the signal, must be real-valued

  • win_len – the size of the window

  • hop_size – the hop size for the stft

  • zp_factor – zero-pad factor. win_len * zp_factor = fft_len

  • window – if None, use Hann window, else, use this as the window

  • centered – use a centered window

Returns

a matrix of shape [num_bins, num_hops] where num_bins = win_len /2 + 1 and num_hops is the total # of hops that “fit” into x

fmplib.unit3.stft_mag(x, win_len, hop_size, zp_factor=1)¶

Return the magnitude of the Short-Time-Fourier-Transform.

Parameters

  • x – the signal, must be real-valued

  • win_len – the size of the window

  • hop_size – the hop size for the stft

  • zp_factor – zero-pad factor; win_len * zp_factor = fft_len

Returns

a matrix where each value is the magnitude of the stft
