fmplib.unit2.calc_energy(x, win_len, hop_size, centered=True)¶

Calculate the normalized energy feature for the input signal.

Parameters

  • x – the signal

  • win_len – window length

  • hop_size – hop size

  • centered – use a centered window

Returns

np.array where each value is the normalized energy of the windowed signal using a rectangular window.

fmplib.unit2.calc_num_frames(x_len, win_len, hop_size)¶

Calculates the number of frames of data resulting when windowing a signal of length x_len, using a window size of win_len and a hop size of hop_size.

Parameters

  • x_len – length of signal

  • win_len – length of window

  • hop_size – hop size (ie, stride length) between windows

Returns

(int) number of frames, i.e., or number of hops

fmplib.unit2.calc_zc(x, win_len, hop_size, centered=True)¶

Calculate the normalized zero-crossing rate for the input signal. At each hop, count the number of zero-crossings in that window and divide by the window length.

Parameters

  • x – the signal

  • win_len – window length

  • hop_size – hop size

  • centered – use a centered window

Returns

np.array where each value is the normalized zc-rate of the windowed signal.
